<h1 align="center">Финальная работа по курсу "Языки веб-программирования"</h1>

## Описание

Веб-приложение является выпускной работой по курсу "Языки веб-программирования". Приложение представляет собой блог, где пользователь может зарегистрироваться, авторизироваться, писать, редактировать и удалять посты.
